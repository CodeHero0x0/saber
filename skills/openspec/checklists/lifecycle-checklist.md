# OpenSpec 生命周期检查清单

## Explore

- [ ] 已记录问题、事实来源、影响范围和未知项。
- [ ] 已检查 Jira 指纹或其他来源是否漂移。

## Propose

- [ ] 已写清范围、非目标、验收标准、依赖和风险。
- [ ] 已识别跨仓顺序、测试证据和任何 L2 外部写入。
- [ ] 已由指定真人批准进入 Apply。

## Apply

- [ ] 仅实现已批准范围，新增假设已回写提案。
- [ ] 已运行相关验证并保存稳定证据引用。
- [ ] L2 操作已有可见预览和确切确认。

## Archive

- [ ] 已记录实际交付、偏差、commit/MR/CI 和验收证据。
- [ ] 已列出遗留风险、后续工作和需要更新的共享资产。
